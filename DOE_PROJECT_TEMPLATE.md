# DOE Project Repository Template

Use this checklist when creating a new DOE project repository.

## Repository Name

```text
DOE-<ProjectName>
```

## Required Files

- `README.md`
- `CONTRIBUTING.md`
- `SECURITY.md`
- `CODEOWNERS`

## Required Folders

```text
models/
datasets/
posters/
presentations/
reports/
```

## README Sections

Every project README should include:

- Project title
- DOE award or project identifier, if shareable
- Principal investigator and project leads
- Repository purpose
- Folder structure
- Project team
- Website or project page
- Data handling notes
- Access and contribution rules

## Artifact Tracking

Each major project artifact should include:

- Artifact title
- Responsible person
- Date or version
- Status
- Link to final artifact or storage location, if applicable
